Visual Studio Code is a powerful, extensible, and lightweight code editor that has become one of the preferred code editors in the Python community.Setup will add Visual Studio Code to your %PATH%, so from the console you can type 'code .' to open VS Code on that folder. You will need to restart your console after the installation for the change to the %PATH% environmental variable to take effect. VS Code provides both Windows user and system level setups.
